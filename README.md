- AI Assistant : Gemini(웹/cli)

# 개발 배경 및 이모저모
- 프로그램 개발 이유
    * 이런거 해보고 싶었고 공부도 할겸해서 개발을 시작함.
    * 니케 게임안에서 스킬 필터링하는 기능을 제공하지 않음. 정확히는 내가 쿼리를 해야하는데 매우 귀찮음.
- 크롤링을 하는 이유
    * 니케에서 공식적으로 api를 제공하지 않음.
    * gemini(웹/cli)가 블라블라링크에 접속해서 정보를 가지고 오는게 제한됨. 지속적으로 업데이트 되서 좀 나아지긴 하는데 프로젝트 시작 할 때는 제한되었음.
    * 정확한 정보와 이미지를 가져오기 위해 블라블라링크 크롤링뿐이라고 판단했음.
- 요즘 서브컬쳐 게임 스킬 분석에 AI가 중요한 이유
    * 스킬 설명이 복잡하고 캐릭터 개수가 많음.
    * 캐릭터 하나하나 스킬 키워드를 분석하기에는 양이 너무 많음. 따라서 텍스트 패턴 분석(구문 분석)을 위해서 AI가 필요함.
    * 근데 AI(LLM)도 직접 분석을 할 생각은 안하고 코드로 딸깍하려는 경향이 있어서 문제임.
- AI Pair Programming
